## ✅ Loop (반복문)

### 1. 중첩 while문
```java
while (조건식 1) {
    반복 실행문 1-a;
    while (조건식 2) {
        반복 실행문 2;
    }
    반복 실행문 1-b;
}
```

🛎️ 실습
```
1일차 1교시입니다.
...
1일차 3교시입니다.
...
5일차 1교시입니다.
..
5일차 3교시입니다.
```
```java
int i = 0;
while (i < 5) {
    int j = 0;      // 위치 중요함.
    while (j < 3) {
        System.out.println((i+1) + "일차 " + (j+1) + "교시입니다.");
        j++;
    }
    i++;
}
//  System.out.println(j);  // 오류가 나는 이유는 추후 설명 예정이지만
                            // scope(범위), global(전역), local(지역)에 해당
```
